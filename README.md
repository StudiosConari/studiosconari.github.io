# Studios Conari Landing Page

Versión actualizada de la landing page de Studios Conari lista para subir con GitHub Desktop.

## Cambios incluidos

- Sección **Trayectoria** rediseñada con **3 banners** en lugar de cards simples.
- **Proyecto Origen** y **Proyecto Resonancia** ahora se muestran como piezas visuales destacadas.
- **VILU y futuras obras** ahora usa un banner tipo teaser con indicación de próximo proyecto.
- Fondos diurno y nocturno incorporados con cambio de tema sin desplazamiento intencional.
- Estructura lista para publicar como sitio estático.

## Archivos principales

- `index.html`
- `styles-v3.css`
- `theme-v3.js`
- `assets/`

## Publicación rápida en GitHub Pages

1. Crea o abre el repositorio local con GitHub Desktop.
2. Copia el contenido de esta carpeta dentro del repositorio.
3. Haz **Commit** y luego **Push origin**.
4. En GitHub, entra a **Settings > Pages**.
5. En **Build and deployment**, selecciona:
   - **Source:** Deploy from a branch
   - **Branch:** `main` / `(root)`
6. Guarda y espera unos minutos.
