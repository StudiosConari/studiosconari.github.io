# StudiosConari_Landing_AlternativaA_Local

Versión local del landing page de Studios Conari con:
- fondo completo visible en toda la página,
- hero sin card inferior ni tags,
- sección del estudio editorial,
- esencia con 3 cards centradas e íconos,
- trayectoria con heading grande + panel de 3 banners,
- marca con emblema de dragón,
- contacto en formato editorial.

Abrir `index.html` en navegador o subir la carpeta completa con GitHub Desktop.

## Actualización de contacto
Las tarjetas de correo, GitHub e Instagram incluyen sus iconos correspondientes a la izquierda y el texto alineado a la derecha, con adaptación para pantallas móviles.
